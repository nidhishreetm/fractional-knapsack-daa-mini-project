# Fractional Knapsack Algorithm using Greedy Method

## Overview
This mini project implements the Fractional Knapsack Algorithm using the Greedy Method in C programming language.

The algorithm maximizes profit by selecting items based on the highest profit-to-weight ratio.

## Features
- Greedy Algorithm Implementation
- Time Complexity Analysis
- Space Complexity Analysis
- Test Case Evaluation
- Modular C Program

## Algorithm Used
Fractional Knapsack Algorithm

### Strategy
1. Calculate profit/weight ratio
2. Sort items in descending order
3. Pick items greedily
4. Take fractional part if needed

## Time Complexity
O(n log n)

## Space Complexity
O(1)
